# Harry Potter Characters

An Angular app displaying Harry Potter characters from the Harry Potter API.

## Name: Ibrahim Yondem - 101525165

## Features
- **Character List**: Browse all Harry Potter characters with images and house affiliations
- **Filter by House**: Filter characters by Gryffindor, Slytherin, Hufflepuff, or Ravenclaw
- **Character Details**: View detailed information including wand type, actor, species, and ancestry
- **House-Specific Colors**: Dynamic color coding based on character house

## Tech Stack
- Angular 17
- Angular Material
- TypeScript
- Harry Potter API (https://hp-api.onrender.com/)

## Running Locally
```bash
ng serve
```
